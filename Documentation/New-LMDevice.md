# New-LMDevice

## SYNOPSIS

Creates a new LogicMonitor device.

## SYNTAX

### __AllParameterSets

```
New-LMDevice [-Name] <string> [-DisplayName] <string> [[-Description] <string>]
 [-PreferredCollectorId] <int> [[-PreferredCollectorGroupId] <int>]
 [[-AutoBalancedCollectorGroupId] <int>] [[-DeviceType] <int>] [[-Properties] <hashtable>]
 [[-HostGroupIds] <string[]>] [[-Link] <string>] [[-DisableAlerting] <bool>]
 [[-EnableNetFlow] <bool>] [[-NetflowCollectorGroupId] <int>] [[-NetflowCollectorId] <int>]
 [[-LogCollectorGroupId] <int>] [[-LogCollectorId] <int>]
 [[-IsPreferredLogCollectorConfigured] <bool>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

The New-LMDevice function creates a new device in LogicMonitor with specified configuration settings.

## EXAMPLES

### EXAMPLE 1

#Create a new device
New-LMDevice -Name "server1" -DisplayName "Server 1" -PreferredCollectorId 123 -Properties @{"location"="datacenter1"}
