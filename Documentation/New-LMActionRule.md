# New-LMActionRule

## SYNOPSIS

Creates a new LogicMonitor action rule.

## SYNTAX

### __AllParameterSets

```
New-LMActionRule [-Name] <string> [-ActionChainId] <int> [-DeviceGroups] <string[]>
 [-LevelStr] <string> [[-Devices] <string[]>] [[-DataSource] <string>] [[-Instance] <string>]
 [[-DataPoint] <string>] [[-ResourceProperties] <hashtable>] [[-Enabled] <bool>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

The New-LMActionRule function creates a new action rule in LogicMonitor.

## EXAMPLES

### EXAMPLE 1

New-LMActionRule -Name "Disk Rule" -ActionChainId 456 -DeviceGroups @("/Servers") -LevelStr "Critical"
