# Get-LMActionRule

## SYNOPSIS

Retrieves action rules from LogicMonitor.

## SYNTAX

### All (Default)

```
Get-LMActionRule [-BatchSize <int>] [<CommonParameters>]
```

### Id

```
Get-LMActionRule [-Id <int>] [-BatchSize <int>] [<CommonParameters>]
```

### Name

```
Get-LMActionRule [-Name <string>] [-BatchSize <int>] [<CommonParameters>]
```

### Filter

```
Get-LMActionRule [-Filter <Object>] [-BatchSize <int>] [<CommonParameters>]
```

## DESCRIPTION

The Get-LMActionRule function retrieves action rules from LogicMonitor based on specified criteria.
It can return a single rule by ID or name, or multiple rules using filters.

## EXAMPLES

### EXAMPLE 1

Get-LMActionRule -Id 123

### EXAMPLE 2

Get-LMActionRule -Name "Disk Remediation Rule"
