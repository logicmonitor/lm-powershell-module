# Set-LMCollector

## SYNOPSIS

Updates a LogicMonitor collector's configuration.

## SYNTAX

### Id (Default)

```
Set-LMCollector -Id <int> [-Description <string>] [-BackupAgentId <int>] [-CollectorGroupId <int>]
 [-Properties <hashtable>] [-EnableFailBack <bool>] [-EnableFailOverOnCollectorDevice <bool>]
 [-EscalatingChainId <int>] [-SuppressAlertClear <bool>] [-ResendAlertInterval <int>]
 [-SpecifiedCollectorDeviceGroupId <int>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Name

```
Set-LMCollector -Name <string> [-Description <string>] [-BackupAgentId <int>]
 [-CollectorGroupId <int>] [-Properties <hashtable>] [-EnableFailBack <bool>]
 [-EnableFailOverOnCollectorDevice <bool>] [-EscalatingChainId <int>] [-SuppressAlertClear <bool>]
 [-ResendAlertInterval <int>] [-SpecifiedCollectorDeviceGroupId <int>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION

The Set-LMCollector function modifies an existing collector's settings in LogicMonitor, including its description, backup agent, group, and various properties.

## EXAMPLES

### EXAMPLE 1

Set-LMCollector -Id 123 -Description "Updated collector" -EnableFailBack $true
Updates the collector with ID 123 with a new description and enables fail-back.
