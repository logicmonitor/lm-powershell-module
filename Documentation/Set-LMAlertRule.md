# Set-LMAlertRule

## SYNOPSIS

Updates a LogicMonitor alert rule configuration.

## SYNTAX

### Id (Default)

```
Set-LMAlertRule -Id <int> [-NewName <string>] [-Priority <int>] [-EscalatingChainId <int>]
 [-EscalationInterval <int>] [-ResourceProperties <hashtable>] [-Devices <string[]>]
 [-DeviceGroups <string[]>] [-Datasource <string>] [-Instance <string>] [-Datapoint <string>]
 [-SuppressAlertClear <bool>] [-SuppressAlertAckSdt <bool>] [-LevelStr <string>]
 [-Description <string>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Name

```
Set-LMAlertRule -Name <string> [-NewName <string>] [-Priority <int>] [-EscalatingChainId <int>]
 [-EscalationInterval <int>] [-ResourceProperties <hashtable>] [-Devices <string[]>]
 [-DeviceGroups <string[]>] [-Datasource <string>] [-Instance <string>] [-Datapoint <string>]
 [-SuppressAlertClear <bool>] [-SuppressAlertAckSdt <bool>] [-LevelStr <string>]
 [-Description <string>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

The Set-LMAlertRule function modifies an existing alert rule in LogicMonitor.

## EXAMPLES

### EXAMPLE 1

Set-LMAlertRule -Id 123 -Name "Updated Rule" -Priority 100 -EscalatingChainId 456
Updates the alert rule with new name, priority and escalation chain.
