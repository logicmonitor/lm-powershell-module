# Set-LMActionRule

## SYNOPSIS

Updates a LogicMonitor action rule configuration.

## SYNTAX

### Id (Default)

```
Set-LMActionRule -Id <int> [-NewName <string>] [-ActionChainId <int>] [-DeviceGroups <string[]>]
 [-Devices <string[]>] [-DataSource <string>] [-Instance <string>] [-DataPoint <string>]
 [-ResourceProperties <hashtable>] [-LevelStr <string>] [-Enabled <bool>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Name

```
Set-LMActionRule -Name <string> [-NewName <string>] [-ActionChainId <int>]
 [-DeviceGroups <string[]>] [-Devices <string[]>] [-DataSource <string>] [-Instance <string>]
 [-DataPoint <string>] [-ResourceProperties <hashtable>] [-LevelStr <string>] [-Enabled <bool>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

The Set-LMActionRule function modifies an existing action rule in LogicMonitor.
When -Enabled is specified, the status sub-endpoint is updated separately.

## EXAMPLES

### EXAMPLE 1

Set-LMActionRule -Id 123 -NewName "Updated Rule" -ActionChainId 456

### EXAMPLE 2

Set-LMActionRule -Name "Disk Rule" -Enabled $false
