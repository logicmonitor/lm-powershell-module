# Remove-LMActionRule

## SYNOPSIS

Removes a LogicMonitor action rule.

## SYNTAX

### Id (Default)

```
Remove-LMActionRule -Id <int> [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Name

```
Remove-LMActionRule -Name <string> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

The Remove-LMActionRule function removes a LogicMonitor action rule based on the specified ID or name.

## EXAMPLES

### EXAMPLE 1

Remove-LMActionRule -Id 123

### EXAMPLE 2

Remove-LMActionRule -Name "MyActionRule"
