# New-LMAlertRule

## SYNOPSIS

Creates a new LogicMonitor alert rule.

## SYNTAX

### __AllParameterSets

```
New-LMAlertRule [-Name] <string> [-Priority] <int> [-EscalatingChainId] <int>
 [[-EscalationInterval] <int>] [[-ResourceProperties] <hashtable>] [[-Devices] <string[]>]
 [[-DeviceGroups] <string[]>] [[-DataSource] <string>] [[-Instance] <string>]
 [[-DataPoint] <string>] [[-SuppressAlertClear] <bool>] [[-SuppressAlertAckSdt] <bool>]
 [[-LevelStr] <string>] [[-Description] <string>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

The New-LMAlertRule function creates a new alert rule in LogicMonitor.

## EXAMPLES

### EXAMPLE 1

New-LMAlertRule -Name "New Rule" -Priority 100 -EscalatingChainId 456
Creates a new alert rule with specified name, priority and escalation chain.
